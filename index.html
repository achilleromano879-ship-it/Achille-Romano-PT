Configura Firebase
🔴
Firebase non configurato
I dati vengono salvati solo in locale su questo browser
📋 COME OTTENERE LE CREDENZIALI
1. Vai su console.firebase.google.com
2. Crea un nuovo progetto (es. "achille-pt")
3. Clicca "</> Web" per aggiungere un'app web
4. Copia i valori dalla configurazione e incollali nei campi qui sotto
5. Nel menu → Firestore Database → Crea database → Modalità test
6. Clicca "Salva e connetti" qui sotto
⚙️ CONFIGURAZIONE
⚡ INCOLLA CONFIGURAZIONE JSON
In alternativa, incolla direttamente il blocco firebaseConfig copiato dalla console Firebase
⚠️ REGOLE FIRESTORE OBBLIGATORIE
Per permettere la lettura e scrittura dei dati, vai su Firebase Console → Firestore Database → Regole e incolla questo codice:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
⚡ Senza queste regole Firebase si scollega ad ogni refresh e i dati non vengono salvati nel cloud.
